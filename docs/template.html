Template

This is a sleek, modern dark/light theme. Use the toggle above or your system preference.

To apply this theme to your docs, include assets/theme.css and assets/theme.js in your page, and add the toggle button with data-theme-toggle.

Example:
<link rel="stylesheet" href="/assets/theme.css" />
<button data-theme-toggle></button>
<script src="/assets/theme.js"></script>
