- `install_models.py`
- `initialize_system.py`
- `kari_dev_cli.py`

- `multitenant_cli.py`
- `server_cli.py`

- `start_production_server.py`
- `stop.sh`
- `system_status_and_fallback.py`
- `wait-for-kari-api.sh`
